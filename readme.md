## 项目概述

* 产品名称：小程序版个人博客
* 项目代号：mpblog
* 服务端源码：https://github.com/yanthink/blog-api

基于 [wepy2.0](https://github.com/Tencent/wepy) 开发。

### 安装

#### 1. 克隆源代码

    git clone https://github.com/yanthink/mpblog.git

#### 2. 安装依赖

    npm install
    
#### 3. 运行编译

    npm run dev
